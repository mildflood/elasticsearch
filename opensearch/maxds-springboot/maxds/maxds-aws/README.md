# maxds-aws

##### To Run: Run as Maven Build by setting value of Goals as spring-boot:run 

The following URLs illustrate how to make different calls to Elastic Search.
* [Find Entity by Name](http://localhost:8080/api/entity/findbyname/microsoft)
* [Find Entity by CIK](http://localhost:8080/api/entity/findbycik/320193)
* [Find Entity by Industry](http://localhost:8080/api/entity/findbyindustry/software)
* [Find Entity by Sector](http://localhost:8080/api/entity/findbysector/service)
* [Find Term Rule by Name](http://localhost:8080/api/termrule/findbyname/Revenues)
* [Find Term Rule by Id](http://localhost:8080/api/termrule/findbytermid/Revt)
